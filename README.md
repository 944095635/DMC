# <strong>DMC 💖 </strong>

- 软件英文名字:   `damn crazy.`

- 软件中文名字:   `绝对是来捣乱的`/`他妈的疯了`

- 是否开源：暂时闭源
- 开发阶段：前期测试阶段UI未定稿


<br></br>
#### 做这个软件的起因

- 某些人为了利益一直不放开自己的权力（让我想起以前的成都公交卡），
- 平时看电视比较多，但是电信机顶盒 和 小米电视，切换系统就要更换遥控器非常的麻烦，快速切换直播源一直是我想要的功能。
- 这个事情早晚会放开的，会有专门的电视App用来播放电视直播，所以这个程序目前只是一个过渡产品，能维持到什么时候也不确定，我甚至不知道是否合法。

<br></br>
